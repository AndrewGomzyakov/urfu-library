## Работа виртуальной машины prolog'a

```prolog
man(a).
man(b).
woman(c).
woman(d).
pair(X, Y) :- man(X), woman(Y).
```
```
?- pair(X, Y)
 (a, c)
 (a, d)
 (b, c)
 (b, d)
```

Предикат `!` - считать унифицированные сейчас значения - окончательными

```prolog
rev(L1, L2) :- rev_(L1, [], L2).

rev_([H|T],L,L2) :- rev_(T, [H|L],L2).
```
