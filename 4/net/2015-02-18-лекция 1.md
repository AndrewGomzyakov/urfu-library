Интернет - набор атономных систем

### Общая структура Интернета
```
[] ~ хост
 |
[]--[]--[] ~ pop, точка присутствия
 \[]--[]/
       |
       |
       [] ~ точка обмена трафика
      /  \
    []----[]
 ```
